# 分账计算器

简单的网页工具，自动按比例分摊课程收入。

**在线访问**：https://coding.chengbei.org/

## 功能

- 总收入填一个数字，自动算出销售提成（× 10%）和扣除三项成本（房屋水电 / 茶歇 / 差旅）后的剩余可分金额
- 剩余按固定比例分给 4 个角色：主讲老师 40% / 技术支持 30% / 课程负责 20% / 课程管理 10%
- 销售明细可选填，按销售姓名自动合并多条记录
- 当前输入自动保存在浏览器本地，刷新不丢
- 历史记录归档：可查看 / 加载回输入区 / 复制 / 删除，同批次名自动覆盖
- 一键复制为 Markdown 表格 / 导出为 PNG 图片
- 单文件 HTML，浏览器双击即用，无需任何安装

## 本地预览

直接用浏览器打开 `index.html`。

```bash
open -a "Google Chrome" index.html
```

## 部署

托管在 Vercel，绑定到 `coding.chengbei.org`（DNS 在 Cloudflare）。

更新发布的步骤见 [CONTRIBUTING.md](./CONTRIBUTING.md)。

## 数据隐私

所有数据只保留在你这台机器的浏览器 localStorage，不上传任何服务器。换设备 / 清缓存就会丢失，重要数据请用"导出图片"或"复制为表格"备份。
